# Go-Tutorial 项目说明

这是一个面向初学者的 Go 语言学习教程仓库，内容按主题划分，每个主题是一个独立的文件夹，包含完整的源代码、PPT、视频讲解和博客文章链接。

---
## 📁 项目结构
go-tutorial/
├── go-defer/
│   ├── src/              # 源代码
│   ├── ppt/              # 幻灯片（.pptx）
│   ├── README.md         # 说明
│   ├── video.md          # 视频讲解地址 (bilibili)
│   └── blog.md           # 博客文章地址（CSDN）
├── ...
└── README.md  
---

## 🎯 使用方式

1. 克隆仓库：
   ```bash
   git clone https://github.com/fcy222fcy/go-tutorial.git
2. 进入你感兴趣的主题文件夹，查看 README.md 获取该主题的学习路径。
3. 阅读源代码、查看 PPT、观看视频讲解，并结合博客文章深入学习。

## 📌 学习建议
1. 先查看主题下的 PPT，理解理论知识
2. 再运行 src 中的示例代码，加深理解
3. 可以参考 video.md 和 blog.md 提供的讲解视频及博客文章
